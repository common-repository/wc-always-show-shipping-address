== Description ==

Using this plugin you can set shipping address fields on the woocommerce checkout page to be always visible. 
Install this plugin and then go to Shop page, add item to cart and go to checkout page. The checkbox will always be selected and shipping fields will always be visible.



== Installation ==

1] Download the plugin zip file.
2] Login into dashboard.
3] Go to plugins->Add New
4] Click Upload Plugin
5] Choose File
6] Click Install Now
7] Click Activate


== Frequently Asked Questions ==

= What are the requirements for this plugin to work? =

This plugin needs the woocommerce plugin to be avctivated on your website.
